Get Started

First, you'll want to check whether scala is installed. From the shell prompt (denoted here by % so don't type it, > denotes the last commant output), type

% scala -version

(-version tells the interpreter to print the version of scala), then press the Enter key. If scala is installed, you will see a message something like the following:

% scala -version
> Scala code runner version 2.11.4 -- Copyright 2002-2013, LAMP/EPFL

If scala is not installed, you can ask your administrator to install it, or you can do it yourself, since scala is free software with no restrictions on its installation or use.

Now, let's play with scala. You can place a scala program directly on the command line using the -e option:

% scala -e 'println("Hello world")'
> Hello World

More conventionally, a scala program can be stored in a file.

% echo "println("Hello world")" > hello.scala
% scala hello.scala
> Hello World
